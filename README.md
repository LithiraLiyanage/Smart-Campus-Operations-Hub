### 🌐 📖 Overview

The **Facilities & Assets Catalogue** module is the **core foundation** of the Smart Campus system.

It enables users to explore, manage, and access all campus resources through a **centralized, intelligent platform** — making resource discovery **fast, structured, and efficient** ⚡

---

### 🎯 🚀 Key Features

✨ **Smart Resource Management**

* 🏫 Lecture Halls
* 🧪 Laboratories
* 👥 Meeting Rooms
* 🎥 Equipment (Projectors, Cameras, etc.)

📊 **Rich Metadata Support**

* 📌 Type
* 👥 Capacity
* 📍 Location
* 🕒 Availability Windows
* 🔄 Status → `ACTIVE` | `OUT_OF_SERVICE`

---

### 🔍 🧠 Smart Search & Filtering
